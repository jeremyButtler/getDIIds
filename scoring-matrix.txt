// This is an example of a scoring matrix you would supply
//   to alnSeq. In this case the scoring matrix is set to
//   the default settings.

// This is a comment and needs to start at the start of a
//   line. It is ok to have stuff after the base
//   combination, since the read in step on looks at the
//   two base pairs and the provided number.  It does not
//   look past the number. However, starting a line with a
//   space is a good way to cause an error.

// A blank line will be ignored so long as it does not
//   have white space.

// Scoring matrix is EDNAFULL * 100. The * 100 allows for
//   an extension penalty of -1 to be -0.01. The aligners
//   will auto divide by 100 when printing out scores, so
//   it works out to a EDNAFULL matrix.
// I think I have the anonymous bases with the correct
//   scores.

// limit for score or gap penalty (not in matrix) is a
//   signed short, so roughly -300000to 30000
